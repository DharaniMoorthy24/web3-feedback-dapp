# 🌐 Web3 Feedback DApp

This is a beginner-friendly Web3 project where users can:
- Connect their Metamask wallet
- Submit anonymous feedback (mock-stored in localStorage)

## 💡 Technologies Used
- HTML, CSS, JavaScript
- MetaMask (Ethereum wallet)
